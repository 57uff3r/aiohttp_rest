.. _installation:

Installation
============

.. currentmodule:: aiohttp_rest_api

Install  with ``pip`` ::

    $ pip install aiohttp-rest-api


The development version can be downloaded from `GitHub
<https://github.com/57uff3r/aiohttp_rest_api>`_. ::


    $ git clone https://github.com/57uff3r/aiohttp_rest_api.git
    $ cd aiohttp_rest_api
    $ pip install -e .
