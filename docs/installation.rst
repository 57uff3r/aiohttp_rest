.. _installation:

Installation
============

.. currentmodule:: aiohttp_rest

Install  with ``pip`` ::

    $ pip install aiohttp_rest


The development version can be downloaded from `GitHub
<https://github.com/57uff3r/aiohttp_rest>`_. ::


    $ git clone https://github.com/57uff3r/aiohttp_rest.git
    $ cd aiohttp_rest
    $ pip install -e .
